Darwin’s Famous Finches:  Part I

Purpose      

 

This activity will explore evolution through the examination of Darwin’s Finches from the Galapagos Islands. We will analyze several different finch species to determine the differences in length and width of the beaks. We will work on the hypothesis that Darwin’s idea of natural selection played a role in the evolution of the finches so they are better adapted to thier environment.

Procedure Part 1:  Measuring Length and Width

 

Setting The Scale:

 

  1. In ImageJ, select line tool from the toolbar. Draw a line over the 11mm reference line from the photo.
  2. From main menu, select Analyze Þ Set Scale
  3. Leave Distance In Pixels at set number Þ type 11 for known Distance Þ Leave Aspect Ratio at 1.0 Þ type mm for Unit of Length Þ Leave Global unchecked.
  4. From main menu, select Analyze Þ Set Measurements.           

 

Measuring the length and width:

 

  1. Now that your scale is set you can start to measure the length and width.

 

  1. On the tool bar select line tool as shown in Figure 1

 

  1. To measure beak length, draw a line from the top of the beak to the tip of the beak as shown in Figure 3.

 

  1. Once the line is draw, select Analyze from the top tool bar.  Then select Measure.

 

  1. A screen will pop up.  You want the number that is under the length column.

 

  1. Record the measurement in data table 1.

 

  1. Repeat steps for measuring width (see Figure 4 as an example).
