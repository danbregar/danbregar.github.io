To calculate the average lichen % cover for a tree species, find the average % cover for each type of lichen (crustose, foliose, fruticose) on each tree.  Then, total those averages for each tree.  Then, divide those averages by the number of trees of each species you looked at.  (1st period only looked at one tree per species; 5th period looked at two trees per species.)
