What is Acceleration?

Physical Science and Technology

 

Acceleration is what occurs any time an object’s velocity is changing.  Since velocity has to do both with how quickly an object is moving and what direction it is moving, acceleration occurs whenever any of the following happens:

1.   An object speeds up

2.   An object slows down

3.   An object changes direction

 

Acceleration is calculated using the following formula:

Acceleration = (Final velocity – Initial Velocity) / Time

a = (vf – vi) / t

 

Acceleration has the units of m/s2.

 

Here is an example of how to calculate acceleration:

A duck is quacking along the road at 6.5 m/s.  Over a period of 6 s, it accelerates to a velocity of 9.5 m/s.  What was the duck’s acceleration?

 

Using the Five Steps:

a.   vf = 9.5 m/s, vi = 6.5 m/s, t = 6 s

b.   a

c.   a = (vf – vi) / t

d.   a = (9.5 m/s – 6.5 m/s) / 6 s

      a = 3 m/s / 6 s

      a = .5

e.   a = 0.5 m/s2
