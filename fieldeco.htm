Field Ecology

 

Field Ecology is an introductory ecology class that emphasizes ecological theory and field methods.  We will spend a great deal of time outdoors, so students should make sure to dress appropriately - there is storage space in the classroom for boots, warm clothes, and rain gear. Our class Showbie code is F92H8.
