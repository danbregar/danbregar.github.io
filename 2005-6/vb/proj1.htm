Project #1 – Tic Tac Toe (Part I)

Visual Basic

 

Your task:

Design a Visual Basic program that will allow two players to play Tic Tac Toe.  The game should allow someone to click on a square – when it is clicked on, it will switch from blank to “X” to “O” and back to blank.

 

There should be four labels that keep score.  One should show how many games “X” has won, one should show how many games “O” has won, one should show how many “Cat’s Games” there have been, and one should show the total number of games played.

 

There should be four command buttons:

“X Wins” should clear the board and add 1 to the total number of games “X” won.

“O Wins” should clear the board and add 1 to the total number of games “Y” won.

“Cat’s Game” should clear the board and add 1 to the total number of “Cat’s Games”.

Each of the above command buttons should also add 1 to the total number of games played.

There should also be an “Exit” button that allows players to quit the game.

 

There are lots of ways to program this – as long as yours works, you will receive full credit.
