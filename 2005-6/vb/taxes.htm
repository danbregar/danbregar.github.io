3rd Project

Visual Basic

 

Write a program that will calculate a person’s taxes.  There should be two forms for this program – a splash screen that shows up initially with the name of the program, your name, and the date you wrote the program (use a “Continue” button to bring up the main form), and the main form used for data entry.

 

The data entry form should have text boxes that allow the user to enter their salary for the year, any tips they received for their job, any investment income they received, and the amount of tax they have already paid.  It should have labels that will show the amount of tax they owe on their salary and tips (called an income tax), the amount of tax they owe on their investment income (called a flat tax), the total amount of tax they owe, and the amount of money they need to pay or will be refunded.  It should have a menu bar with a File menu and a Help menu.  The file menu should have two options – “Find Tax” and “Exit”.  The help menu should have one option – “About”.

 

Use a .bas code module to hold the following functions:

1.  FindTax – This function should take two arguments:  a currency number that represents the amount of money being taxed, and an integer that represents the type of tax being used.  If the integer is a 1, the tax is an income tax.  Income taxes are 5% on the first $10,000, 10% on the next $25,000, and 20% on any amount above $35,000.  If the integer is a 2, the tax is a flat tax.  Flat taxes are 20% on the entire amount.  This function should return a currency number that represents the tax owed.

2.  RefundDue – This function should take two currency numbers as arguments.  The first argument should be the amount of tax a person owes.  The second argument should be the amount of tax a person has already paid.  This function should return a Boolean value of True if the person has already paid more than he or she owes.  It should return a Boolean value of False if the person owes more money than he or she has already paid.

 

When the user chooses “Find Tax” from the File menu, the program should use the FindTax function to calculate the amount of income tax on the salary and tips combined.  It should also use the FindTax function to calculate the amount of flat tax on the investment income.  It should display the income tax and flat tax in the appropriate labels.  It should then use the “RefundDue” function to determine whether the user will be getting a refund or will need to pay additional tax.  If the user will be getting a refund, the label for the amount of money they need to pay or will be paid should change to say “Refund:”.  If the user will need to pay additional tax, the label for the amount of money they need to pay or will be paid should change to say “Tax-to-pay:”.  The program should then display the amount of money the user will either be refunded or will need to pay in the appropriate label.  Make sure this amount of money is always displayed as a positive value.

 

When the user chooses “Exit” from the file menu, the program should close.  When the user chooses “About” from the Help menu, the splash screen should be displayed.

 

Save all the files for your program on your H: drive in a folder called “Project3” so they will be easy to transfer.

 

You will be graded using the following criteria:

a.       Properly formatted variable names and object names. (5 points)

b.       Properly commented code and variables. (5 points)

c.       Proper user interface – tabs, Escape, and Enter work as expected. (5 points)

d.       Proper code functioning – the program works.  (25 points)
