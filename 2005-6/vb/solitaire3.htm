Solitaire, Part III

Visual Basic

 

For the third part of your solitaire program, you will set up the user interface so your game looks like a real solitaire game.  You can use the Microsoft Solitaire game to figure out how your game should look.  Each card should start in the correct place, and there should be four blank spots for aces at the top right of the screen.  In addition, cards that aren’t dealt yet should be stacked up, face-down, in the upper left corner of the screen.

 

Each time you click on one of the cards that hasn’t been dealt yet, it should flip over.  When you reach the end of the deck, those cards should all flip over and start again (just like Microsoft Solitaire when set to “Draw One”).
