Project #2 – Message Machine

Visual Basic

 

Design a program that will allow the user to use icons to display a message.  The program should have two image boxes that allow the user to display any one of three images by clicking the images.  (Both image boxes should have the same three images; the user can change the current image by clicking on the image box).  There should be a “Display” command button that allows the user to display a message in a label.  When the images in both image boxes match, clicking the “Display” button will cause the label to contain a message corresponding to that image.  When the images in both image boxes are different, clicking the “Display” button should cause the label to clear and a message box to appear saying “Please select the same image in both image boxes.”  There should also be an “Exit” button allowing the user to quit the program.  The “Display” button should be the default button and the “Exit” button should be the cancel button.

 

Add to this program a charge to display the messages.  Use a label to display the current amount of money deposited (start with $0.00).  Use a module-level variable to store the amount of money deposited.  Each time the user clicks “Display”, if the current amount of money is greater than 0 and both images are the same, the program should display the message and subtract $1.00 from the amount of money deposited.  If there is no money left, display a message box saying “Please add more money”.  If the amount of money is 0, no message should be displayed in the label.  Users should add additional money by selecting an amount from an option button ($1, $5, or $10) and clicking a “Deposit” command button.

 

You will be graded based on the following criteria:

a.   Properly formatted variable names and object names. (5 points)

b.   Properly commented code and variables. (5 points)

c.   Proper user interface – tabs, Escape, and Enter work as expected. (5 points)

d.   Proper code functioning – the program works.  (15 points)
