Solitaire (Part II)

Visual Basic

 

Now that you have 52 images on the screen, you need to be able to shuffle them.  Make a command button on the screen that can be used to randomly shuffle the cards.  You should be able to shuffle the cards whether they are face up or face down, and the images should be updated to show the correct card.

 

Here are some hints for shuffling:

  1. You’ll need to generate a card randomly by picking a number between 1 and 52.
  2. You’ll need a “master list” of cards – these can be stored in a variable array that is separate from the deck of card that appears on the screen.
  3. You’ll want to keep track of which cards have already been assigned – you can make a variable array to do this.
  4. You’ll want to make sure that you don’t assign a value to a card that’s already been assigned another value!
