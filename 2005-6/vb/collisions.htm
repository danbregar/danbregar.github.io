Collisions Project

Visual Basic

 

For this project, you will need to write three completely separate programs.  Each will accomplish the same task, but in three different ways.

 

Your goal is to write a program that will allow you to control one of four command buttons on the screen.  Each command button should be a different color and have a number on it.  Pressing the numbers “1” through “4” should shift control from one button to another.  The arrow keys should move whichever button is currently being controlled.  The buttons should not be allowed to move off the edge of the form.  Whenever two visible buttons have the same “top” and “left” properties, both of the buttons should disappear.  When all the buttons disappear, a message box should appear that says “You Win!” and the program should start over.

 

You need to write three separate programs to accomplish this task.  The first program should not use control arrays or functions.  The second program should use one control array for all four buttons, but no functions.  The third program should use one control array for all four buttons and it should also use a function called “Collisions” which takes two objects as parameters and returns a Boolean value.  The “Collisions” function should look at the two objects it is given and return True if the two objects have the same “top” and “left” properties.
