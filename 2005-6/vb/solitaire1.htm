Project:  Solitaire 1

Visual Basic

 

This project is a very complex project that will involve several different steps.  Here are your first few tasks:

 

1.                  Create a control array of 52 images.  Each image should have a “tag” property that indicates its rank and suit (for example, “QH” could be queen of hearts).

2.                  Each image should show up “face down” – in other words, the value of the card should not be visible.

3.                  Clicking on a card should cause the card to become visible.

4.                  Clicking and dragging a card should cause the card to move to a new spot on the form.
