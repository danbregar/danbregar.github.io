Final VB Project – Memory

Visual Basic

 

Design a program that will allow one user to pick a sequence of four colors.  After this sequence has been chosen, it should be hidden from view.  A player will then try to guess the sequence.  Each time they guess, the program will give them two pieces of information:  The number of colors they guessed right but are in the wrong spot, and the number of colors they guessed right and are in the right spot.  The player should have ten chances to guess the colors right – if she gets them right by the 10th try, she wins, otherwise, she loses.

 

Start by designing the user interface – it should retain a list of the right color / right spot and right color / wrong spot numbers along with each guess the user has made.

 

Then, write code that will allow the first player to pick, store, and hide the color sequence.

 

Then, write code that will allow the second player to pick and guess a color sequence.

 

Then, write code that will identify the number of guesses that are the right color and the right spot.

 

Finally, write code that will identify the number of guesses that are the right color but the wrong spot – this part of your program will be extremely tricky!  I will give you several hints over the next couple of weeks.
