Three Cows

Visual Basic

 

 

This is a problem that is often more difficult than it seems …

 

Write a program with one command button.  When the command button is pressed once, one image of a cow should appear.  When the command button is pressed twice, another cow should appear.  When the command button is pressed a third time, another cow should appear.  When the command button is pressed again, all three cows should disappear.

 

Have fun!
