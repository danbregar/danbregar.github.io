Timberhill Plant Web Page Checklist

Field Biology

 

For each student you are assigned, check the following:

 

0  A folder on Student File Server named “The_Plant” – with the common name of the student’s plant, with each word capitalized, and separated by spaces.

0  In this folder, there should be a 8” wide, 100 dpi picture of the plant saved in JPEG format.  The name of this file should be “The_Plant.jpg”.

0  In this folder, there should be a 5” wide, 72 dpi picture of the plant saved in JPEG format.  The name of this file should be “The_Plant-5.jpg”.

0  In this folder, there should be a 1” wide, 72 dpi picture of the plant saved in JPEG format.  The name of this file should be “The_Plant-icon.jpg”.

0  In this folder, there should be a web page (created in Microsoft FrontPage or Microsoft Word).  The name of this file should be “index.htm”.

 

Also check to make sure that the student’s web page contains the following components:

0  The local name (as used at Timberhill) of the plant.

0  The 5” photo of the plant.

0  The name of the student and the date the web page was created.

0  The full scientific name (spelled correctly!) and additional common names for the plant.

0  Important and/or interesting data from USDA PLANT database (cited correctly!)

0  Links to at least three additional internet sources (use Google).  Each source should include a couple of sentences describing the type of information someone would find at the site.

0  Scanned or transcribed references from Hitchcock, Gilkey, and/or Haskin (cite cite cite!)

0  The web page should be formatted so that all text is centered.  The font should be Times-Roman.  The local name of the plant should be font size 24 and bold; the rest of the information should be in font size 16.
