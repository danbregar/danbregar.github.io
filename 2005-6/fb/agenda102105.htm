Field Biology (1st Period)

 

You will have 15 minutes to take the plant quiz today.  During the other 30 minutes of class, here is your task:

 

1.   Read through the list of questions posted here that relate to the Noyes property. 

2.   With a partner, pick any five of these questions and check to make sure that:

a.   The question has is examining the relationship between two elements

b.   There is an adequate measuring word associated with each element

c.   The question is something that would likely be able to be answered given the time, material, and logistical constraints of this class

3.   If necessary, rewrite these questions so that they meet criteria a, b, and c above

4.   Pick ONE of the questions that you looked at and do some background research:

a.   See if you can find, using Google, a scientifically accepted protocol for making each measurement described in the question.

b.   See if you can find any research that has already been done that pertains to either or both of the elements in the question

c.   Jot down any information that you find and include the URL of the web site where you found the information
