Reinventing the Wheel Reflection

Physical Science and Technology

 

Please answer the following questions on a separate sheet of paper.  You should work individually on these questions; every student should turn in their own copy.  Make sure to write your name on the paper for full credit.

 

1.   Describe one part of the project that you found either easier or more difficult than you expected.  How did you expect to complete that part of the project?  What did you end up doing to complete that part of the project?

 

2.   If you needed to complete this project again, with a different distance and a different number of rotations, would you be able to do it more quickly this time?  What would you know that would allow you to save time?

 

3.   If you had to give someone written instructions to create a wheel that would travel a certain distance in a certain number of rotations, what would you give them?   Try to make the instructions as simple as possible while still working correctly for any distance and any number of rotations.

 

4.   Describe one way in which you and your partner worked well together in order to successfully complete this project.

 

5.   Describe one way in which you (personally) will try to improve your ability to work in a small group on the next project we do in PST.
