3.      The non-scientific definition of work just means that someone is doing something – a job, homework, chores, etc.  It also means that something is difficult (this problem set is hard!).  The scientific definition of work specifically refers to the amount of energy that is used to change the motion of an object.

 

4a.    a)   t = 16 s; vf = 18.3 m/s; vi = 0 m/s; m = 12.88 kg; d = 89.5 m

         b)   a

         c)   a = (vf – vi) / t

         d)   a = (18.3 m/s – 0 m/s) / 16 s

               a = 18.3 m/s / 16 s

               a = 1.14

         e)   a = 1.14 m/s/s

4b.    a)   Same as above; a = 1.14 m/s/s

         b)   F

         c)   F = ma

         d)   F = (12.88 kg) (1.14 m/s/s)

               F = 14.68

         e)   F = 14.68 N

4c.    a)   Same as above; F = 14.68 N

         b)   W

         c)   W = Fd

         d)   W = (14.68 N) (89.5 m)

               W = 1313.86

         e)   W = 1313.86 J

 

5a.    Force is a push or a pull on an object

5b.    Work is what happens when energy is used to change the motion of an object

5c.    Energy is “the currency of physics” – like money, it can be stored or it can be exchanged (transformed).  One example of an exchange is work – work can happen when energy is transformed from one form to another.
