MOTION’S MYSTERIES

Physical Science and Technology -- Motion Lab

 

 

If you wanted to describe to someone how an object moves, how would you do it?  This series of activities is designed to show you the proper manner in which the motion of an object is communicated.  Key terms you will eventually need to understand include the following:  distance traveled, displacement, speed, velocity, and acceleration.

 

 

ACTIVITY 1:  MOTION PREDICTION

 

If you wanted to describe to someone how an object moves, how would you do it?  Often, graphs are used to describe motion.  Below are two graphs.  In the first graph, the displacement of an object has been plotted against time (d-vs-t).  Take some time to study this graph.  Look at the labels on each of the axes.

 

 

1.  On a piece a paper, describe in words the motion of the moving object shown in this graph.  If you do not understand all of the terms on this graph or if you are unsure, that is okay.  In this activity, we want you to determine what you already know about motion.

 

In the second graph, the velocity of an object has been plotted against time (v-vs-t).  Take some time to study this graph.  Look at the labels on each of the axes. 

 

 

2.  On a piece a paper, describe in words the motion of the moving object shown in this graph.  If you do not understand all of the terms on this graph or if you are unsure, that is okay.  In this activity, we want you to determine what you already know about motion.
