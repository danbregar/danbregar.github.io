Project #1 – Personal Web Site Home Page

Web Design

 

In this project, you will use all of the techniques you have learned in Web Design so far in order to create a personal home page.  Your home page should include the following components:

 

1.   An attractive color scheme.

2.   A design (using tables or any other means) that creates an attractive visual design.

3.   At least 5 text hyperlinks.

4.   At least 5 images; at least one image needs to be a hyperlink.

 

If you’d like, you can also create links to additional pages that you’ve created that are designed to be part of an entire personal web site.  These pages should share a similar layout and design as your home page but should be for specific sub-topics (for example, your musical or athletic activities).  This is not required for this project, but you will be working on these pages after your home page is complete.
