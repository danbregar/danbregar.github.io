Web Design

Assignment #4

 

The purpose of this assignment is to help you choose a color palette for your website.

 

Part A:

Choose a six-color palette for your personal web page by using the tool on the following page:  http://www.colormatch.dk/

 

Try to choose colors that are visually appealing, relatively subtle (unless you REALLY want your site to be irritating!).  You can surf the web to find some other sites that have color schemes that you think look good.

 

Write down the six-character code for each of the colors so you don’t forget what they are!  (Each code is a precise description of one color.)

 

Part B: 

Use these colors in your personal web page.  Try to make sure that all text is readable, and that your backgrounds and fill colors are not overpowering.

 

To select one of these colors, open up any of the color dialog boxes (choose “Font” from the “Format” menu, then select “More Colors” from the “Color” drop-down menu).  In the space labeled “Value”, type the six character code for the color you wish to use.

 

Once you’ve used a color in your web page, it will show up in the color drop-down menus under “Document Colors”.
