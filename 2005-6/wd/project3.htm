Project #3 – Teacher Web Site Outline Pages

Web Design

 

For this assignment, you will complete the layout and framework for your Teacher Web Site pages.  At this point, you do NOT need to include the specific details that will come from your interview later this week or next – but all of the other information you will use to create your teacher web site should be there.  Specifically, your page should include:

 

1.   Headings for each page and the sections on each page

2.   A consistent color scheme applied to each page in the web site

3.   Consistent fonts and font sizes for each page in the web site

4.   Tables (or layers) that establish the layout for each section on each page

5.   A banner frame (or in-line frame)

6.   A menu frame (or in-line frame) with active links for each page on the site

 

Since you are working in groups to make your teacher web sites, you will need to consolidate your pages onto one student’s H: drive.  This assignment is due on Friday, April 7 – on that day, along with making sure all your pages are on one student’s H: drive, you should turn in a piece of paper with all your names on it, the name of the teacher whose web site you are designing, and the name of the student whose H: drive you have copied your pages.  If you need help copying your files from your H: drive to another student’s H: drive, please let me know so I can help get those files transferred.
