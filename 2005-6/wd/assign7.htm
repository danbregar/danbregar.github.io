Assignment #7

Web Design

 

This assignment is designed to help you develop your personal web site into a cohesive set of pages that share a common color scheme, layout, and font selection.  When you are through with this assignment, you will have a completed web site that can be uploaded onto the internet for public viewing.

 

1.   Look at the following web pages:

http://www.radioparadise.com

http://www.apple.com

http://www.adobe.com

http://www.cleverdesign.com

http://www.leveltendesign.com/

 

      For each of these sites, jot down some notes on the site’s color scheme, overall layout, font choice, and font size.  Think about how the similarities and differences between these sites and your personal web site, and use that information to help you make final choices for your personal web site.

 

2.   Either by using a drawing (with notes) or by setting up a template, document your choices for your final color scheme, layout, and font details for your personal web site.  All of the pages you design for your personal web site should share all of these features – in other words, each page should have the same color scheme, layout, and fonts!

 

3.   Create a new folder in your WD folder called “Final Personal Web Site”.  If you want to keep some or all of your existing personal home page, please copy and paste the appropriate files into this new folder.

 

4.   Your final personal web site should contain the following elements:

A.  At least five different web pages (all of which share the same color scheme, layout, and fonts!)

B.   A navigation bar / menu bar (this can be down the left-hand side of the screen or anywhere else you choose) that takes the viewer between the pages you are designing

C.  Appropriate images and content (please note that you should not use text or graphics that might be offensive to others – and also please do not link to sites that use text or graphics that might be offensive to others!)
