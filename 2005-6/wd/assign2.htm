Web Design

Assignment #2

 

The purpose of this assignment is to give you practice working with tables and links in Microsoft FrontPage.  Make sure you save your document often (to your H: drive) so that you don’t lose your work accidentally!  You can preview your web page at any time by saving it, then by opening the file in Internet Explorer by choosing “Open” from the “File” menu.

 

Part A:

Open up Microsoft FrontPage on your computer, and open the personal web page that you created the other day.  Follow these steps to create a table that you can use to visually organize your page:

 

1.   From the “Table” menu, choose “Insert … Table”.  Make sure you create a table with at least three rows and three columns.  You can adjust any of the other settings as you wish.

 

2.   In each cell of the table, type a quick summary of some of your favorite web pages (for example, Yahoo or Google News).  You can organize and arrange the information however you want. 

 

3.   Experiment with resizing columns and rows – point your mouse to the lines of the table and click and drag to resize.

 

4.   Experiment with modifying specific formatting options for each cell by choosing “Table Properties … Cell” from the “Table” menu.

 

5.   Experiment with modifying formatting properties for the entire table by choose “Table Properties … Table” from the “Table” menu.  For example, you can eliminate the raised table borders by changing the options in the “Borders” section.

 

Part B: 

Add links to the text in your table by following these steps:

 

1.   Highlight the text you wish to turn into a link.

2.   From the “Insert” menu, choose “Hyperlink”.

3.   In the address box, type the entire web address (including “http://”) of the web site you wish to link to, then click “OK”.

 

Add a link for each cell in your table.
