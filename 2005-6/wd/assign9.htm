Assignment #9 – Teacher Interview

Web Design

 

For your next project, you will work in groups of three to design a web site for a teacher.  The web site will have four pages – working together, you will create a main / intro page that talks about the teacher.  Then, each of you should tackle one of these additional pages:

 

1.   General course information for the teacher – this page should describe the different courses the teacher teaches.

 

2.   Assignments – this page should have an up-to-date listing of all assignments the teacher has given this semester, along with the date assigned and due dates.  Wherever possible, this page will also include links to electronic versions of the actual assignments.

 

3.   Calendar – this page should have an up-to-date listing of all the class activities that have taken place so far this semester along with a description of future activities.

 

Today, you should accomplish three tasks:

 

1.   Find two other people in the class you can work with on the web site.

2.   With your group, come up with three possible schemes for fonts, layout and colors for the web site.

3.   Find a teacher in the school that you can work with on this site – for now, you do not need to ask the teacher any specific questions except whether he or she is interested in having you make a web page.  Please do not interrupt teachers while they are teaching a class!  (It is OK to go into a teacher’s classroom during class if the students are working independently, though).

4.   Using FrontPage, create a “mock-up” – a quick version of the web site that shows the layout, color scheme, and font choices.  The mock-up does not have to have any real information – it can simply be text you make up (for now).  Later, you will replace it with actual information you get by interviewing the teacher.
