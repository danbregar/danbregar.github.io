Final Project – Class Web Site Competition

Web Design

 

Your final project for this class is to create a web site that will allow all of the students in the Web Design class to showcase their work for the semester.  At the end of the year, the class will choose one web site to that will be used as the official class site.  You should work in groups of 2 – 3 to create a web site with four main pages (seniors should work in groups with other seniors, since they will have less time to complete this project):

 

1.   An index.htm page that describes the class and each of the major assignments

2.   A Personal Web Site page that contains links to each student’s personal web site

3.   A Teacher Web Site page that contains links to each student’s teacher web site

4.   A Community Organization Web Site page that contains links to each student’s community organization web site

 

General requirements for all pages are:

1.   Each page should be created by one student, and that student’s name and the revision date should appear on the page

2.   All pages should share a common color scheme, layout, menu bar, banner, and font

3.   Inline frames should be used (no “regular” frames at all!)

4.   Pages should be visually appealing with a “modern” look (e.g., sans serif fonts such as Arial should be used and layouts should conform to current standards)

5.   Images and digital photos should be used liberally throughout the site (if you have a digital camera, you may use it; there will be one or two school cameras available to you as well)

6.   Hover buttons should be used as menu buttons for linking to the four main site pages

 

Assignment for Day 1 (5/30/06):

1.   Choose your partners

2.   On a sheet of paper, write your name and the names of your partners.

3.   On your paper, describe / draw at least three different layouts that you could use for your web site.  Include information about color schemes, fonts, location of images / graphics, and the location of menus, buttons, and hyperlinks.

4.   On your paper, describe the information you will include on each of the pages that would make up the web site, along with who would be responsible for these pages.

5.   With your partners, begin creating a “template” page – a web page that has the right color scheme, font scheme, and layouts so that you can all make web pages that have the same look and feel.

6.   Turn in your paper to the instructor at the end of the period.

 

Assignment for Day 2 (6/1/06):

1.   Talk to the other students in the class and record:

a)   How to spell each student’s name

b)   Which students worked together on the teacher web pages and which teacher they worked with

c)   Which students worked together on the community organization web page and which organization they worked with

2.   For each of the pages you are making, make a list of all the photos and/or images you will need.  Describe these images in enough detail that you or any of your partners could figure out what they should look like.

3.   Start taking photos and compiling images for use on your page.  If you have your own digital camera, it would be helpful if you brought it to class for the next few days – if not, you can share one of the science department cameras (images can be loaded by attaching the camera to any Windows XP computer).

4.   If you have time, you can begin adding images to your pages.  Remember to resample your images so they take up as little memory as possible.

 

Assignment for Day 3 (6/6/06):

1.   Add images and pictures to all your pages in preparation for creating hyperlinks to each personal web page, teacher web page, and community web page

2.   Make any changes you wish to your personal web page, teacher web page, or community web page – final uploads of these pages will take place on Thursday during A block

3.   Make sure you resample all images and photos on your class web pages – digital photos can take up tons of room and take a long time to download!

 

Assignment for Day 4 (6/8/06):

1.   Complete the layout of your class web pages – all text, images, and photos should be in the right spot.

2.   On a sheet of scratch paper, write down the names of the students in your group and the exact location of your “index.htm” file for your class web site (make sure you note whose H: drive the file is stored on).

3.   If you have been working on a flash drive or CD-ROM, make sure to copy the latest revision of your work to the school network.
