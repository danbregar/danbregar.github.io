Assignment #5

Web Design

 

The purpose of this assignment is to help you add images and graphics to your web page and use those elements as hyperlinks.

 

To add an image to a Microsoft FrontPage document, use the following steps:

1.   Make sure you are adding the image to a table.  FrontPage will let you add an image anywhere in a document, but if you place it in a table, you will have much more control over its positioning on the page (including how it sits with the text around it).

 

2.   Click on the table cell where you want to add the image.  Choose “Picture” from the “Insert” menu; for now, you can choose any of the image categories you wish.  If you have an image saved on your hard drive (for example, an image downloaded from another web page), choose “From File”.

 

3.   To change the size of the picture, you can either click on it and drag the black dots, or you can right-click on it and choose “Picture Properties” and manually change the width and height.

 

4.   Once you have resized the pictures, right-click on it and choose “Resample” – this will make the file smaller and improve its appearance.

 

To add a hyperlink to an image, click on it, then choose “Hyperlink” from the “Insert” menu.  Type the name of the web page you wish to link to.

 

Today, you should add images and graphics to your personal web page.  Put at least 5 images – not all of them need to be hyperlinks, but some should.  Experiment with changing the row and column sizes of tables in order to make the text on your page flow well with respect to your images.  (For example, try making text flow around your image so that the image appears to be sitting in a circular placeholder – this will require a lot of rows and columns in your table to pull off correctly!)
